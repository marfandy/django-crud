# django-crud

Prerequisites

- python >= 3.8

1. Clone this project

   ```bash
   git clone git@github.com:marfandy/django-crud.git
   ```

2. Create `.env` and set your environment variables

   ```bash
   python3 -m venv venv
   ```

3. Install packages

   ```bash
   pip install -r requirements.txt

4. Start App!

   ```bash
   python3 manage.py runserver
   ```
